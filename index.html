MySQLの導入とか

インストール済みの場合は3から！
  • 1. Homebrewの確認
    brew --version
    確認できない場合はコチラ
  • 2. インストール
    brew install mysql
  • 3. mysqlサーバーの起動
    mysql.server start
  • 4. mysqlサーバーにログイン
    mysql -uroot
    （ユーザ設定などが必要なら各自で...）
  • 5. ログアウト
    quit
  • 6. mysqlサーバーの停止
    mysql.server stop
